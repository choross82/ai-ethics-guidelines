- **국가기관별**: 국제기구·정부가 공식 채택 또는 발행한 AI 윤리 문서
- **교단별**: 종교 단체·교단·교회 협의체가 공식 채택한 AI 윤리 가이드라인·성명·결의안·교령

## 참고사항

- 원문은 영문 또는 해당 국가 언어이며, 요약.md는 한국어로 작성
- 교단별 자료는 가족별(가톨릭 → 에큐메니컬 → 장로교 → 성공회 → 복음주의 → 기타 → 유대교 → 초종교)·연대순 정리
